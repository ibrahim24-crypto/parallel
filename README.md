# Next.js 3D Printer App

A stylized 3D printer visualization built with Next.js and Three.js.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Interactive 3D printer model
- Auto-rotating view with orbit controls
- Stylized design with vibrant colors
- Text: "hello i am a 3d printer"
- Responsive and smooth animations
